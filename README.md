# 🎮 Tic Tac Toe Game

An interactive browser-based Tic Tac Toe game built using HTML, CSS, and JavaScript.  
Play with a friend and enjoy smooth gameplay with instant win detection.

---

## 🚀 Live Demo
👉 https://tic-tac-toe-20f04.netlify.app

---

## 📸 Features

- 🎯 Two-player gameplay (X vs O)
- 🧠 Win detection logic
- 🔄 Restart game functionality
- 🎨 Clean and responsive UI
- ⚡ Smooth interactions

---

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript
